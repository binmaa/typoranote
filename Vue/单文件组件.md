* 单文件组件开发注意



* 模块、库使用

1. 下载

   ```shell
   npm install axios
   #缩写
   npm i axios
   ```

2. 导入

   ```javascript
   import axios from 'axios'
   ```

+ 组件封装使用

```javascript
<template>
  <div>
      <player></player>
  </div>
</template>

import player from "./components/player.vue"
export default {
    //注册
    components:{
        player//标准写法player:player
    }
}
```
